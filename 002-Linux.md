## Carpetas de sistema de Linux:

- **bin** - guarda archivos binarios del sistema - ejecutable
- **boot** - contiene el sistema de gestión del arranque - no lo toques
- **cdrom** - carpeta virtual cuando insertamos un cdrom dentro del ordenador
- **dev** - devices - contiene carpetas virtuales para cada componente de hardware del ordenador
- **etc** - carpeta destinada a guardar la configuración de los programas
- **home** - carpeta personal de cada usuario de la máquina - aquí es donde guardáis vuestros documentos
- **lib** - librerías (software compartido por otros softwares)
- **lost+found** - oficina de objetos perdidos
- **media** - donde se montan los dispositivos, cdrom, pendrives, etc
- **mnt** - montar discos duros externos, o unidades de disco gruesas
- **opt** - opciones de aplicaciones
- **proc** - los procesos del sistema aparecen como archivos - concepto heredado de UNIX, donde cada proceso o cada dispositivo es un archivo
- **root** - es la carpeta personal del superadministrador - no tocar
- **run** - archivos ejecutables
- **tmp** - carpeta de archivos temporales
- **usr** - programas instalados por el usuario
- **var** - archivos variables de configuración de programas - por ejemplo todo lo de apache está en var
